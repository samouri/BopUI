# Bop Music #

## Directory Structure ##

### BopUI/server ###
All things related to the nodejs server.  Main responsibilities include:
* Serving index.html for react
* Acting as a server for api calls from react
* Auth
* MongooseDB Saving

### BopUI/shared ###
Code that may be shared between server/client

MongoDB Models

### BopUI/app ### 
    Reactjs Webapp
